    • Metoda GET - zwraca całą listę zadań zapisanych w bazie danych TODO.
      • Bazowa metoda, która zwraca dane z bazy danych przy przejściu na strone API.

      Do wywołania metody wystarczy przejść na stornę przez link
      lub przy użyciu aplikacji służącej do zarządania endpointami
      takiej jak: Postman i przesłać żądanie GET.

    • Metoda POST - pozwala dodać do bazy danych zadanie.
      • Do przesłania danych przez API potrzebna jest aplikacja taka
      jak wyżej wymieniony Postman i przesłać żądanie POST
      z odpowiednimi argumentami. Przykład żądania: 

          {
        "czynnosc": "Przykładowa czynnosc",
        "data": "2022-01-01",
        "godzina": "21:22:00",
        "opis_czynnosci": "Opis co trzeba zrobić w danej czynnosci",
        "priorytet": "medium"
    }
      Po prawidłowym przesłaniu żądania otrzymamy komunikat:
          {
        "info": "Dane dodane poprawnie",
        "success": true
    }
